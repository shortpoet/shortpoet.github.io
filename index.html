Summary

Methodology and Analysis

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to assemble on over 500 cities. These cities were selected at random from a dataset of 547 cities (cities.csv).

After assembling the dataset, I used Matplotlib to plot various aspects of City Latitude vs. Weather. Factors I looked at include: temperature, cloudiness, wind speed, and humidity. Additionally, I filtered the data for cities with Low Temperatures yet Low Latitudes with corresponding Overlay Map and dataframe. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
